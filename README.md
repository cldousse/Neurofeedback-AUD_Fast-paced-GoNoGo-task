# Cued Go/No-Go Task — AUD Neurofeedback Experimental Study

Ongoing research at CHU Brugmann (Brussels, Belgium) investigating the effects of SMR neurofeedback training on attentional biases and inhibitory control in patients with Alcohol Use Disorder undergoing detoxification.

## Description
This repository contains all files necessary to run the experimental cognitive **Cued Go/No-Go Task**, used to assess cognitive functions as part of this study.

## Contents
- **Main script** (`.py` file) — Python code to run the experimental task.
- **Resource files** — Images (`.jpg`, `.bmp`, `.png`) used in the task.
